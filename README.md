# geosy
Geo processing doesn't have to be so hard

## Setting up for development

#### Local

##### Prereqs

* Python3.8+
* venv
* pip
 

##### Installing dependencies in a virtualenv
    $ python3 -m venv venv
    $ source venv/bin/activate
    $ pip install -r requirements.txt

#### Running tests
    $ cd tests
    $ python -m unittest
